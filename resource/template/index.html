CloudSSH

Manage your SSH private key in the cloud, you can log in to the server on the PC and mobile terminal for maintenance, liberating your productivity.

Account

Simple

  • SignUp:cssh signup -s https://ssh.example.com
  • Login:cssh login -s https://ssh.example.com -u hi@example.com -p thisispassword
  • Logout:cssh logout -f
  • Help:cssh help
Get started

Server management

Direct

  • Create:cssh server create
  • List:cssh server list
  • Delete:cssh server delete --id="1,3,5..."
  • Dial:cssh server dial -i 1
Hosted instance

Organization

Share

  • Create:cssh org create
  • Add User:cssh org -o1 user add
  • List Server:cssh org -o1 server list
  • Edit Server:cssh org -o1 server edit -i1
  • Dial Server:cssh org -o1 server dial -i1
Self-Host

You can directly use the server instance we maintain, just download the client and execute a few simple commands.
Start here now: cssh signup -s -u hi@example.com -p thisispassword
Useful help commands:
   cssh help
   cssh server help
   etc.
